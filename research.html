Definition of Online Media

Online media, alternatively termed digital media or digital content, encompasses all forms of information and content generated, disseminated, and consumed via the internet or digital platforms. This broad category includes websites, blogs, news articles, videos, podcasts, online magazines, various social media platforms, and other digital formats.

Class 2

Platform Architecture and Infrastructure

The platform's foundation lies in its architecture, which, in turn, relies on a robust infrastructure. The internet serves as the overarching framework, employing a client/server model with various interconnected components. This hierarchical structure ensures seamless functionality and interaction within the platform.

Client/Server Architecture

When you enter a URL in a browser:
1. The browser parses the URL to extract details like the protocol and domain.
2. It resolves the domain to an IP address using DNS.
3. A TCP connection is established with the server.
4. An HTTP(S) request is sent to the server, specifying the desired resource.
5. The server processes the request and sends back an HTTP(S) response with the requested resource.
6. The browser renders the HTML content, fetches additional resources, and displays the fully rendered webpage to the user.

Client/Server Infrastructure

Client/server infrastructure divides computing tasks between clients (end-user devices or applications) and servers (powerful systems). Clients initiate requests, while servers respond by providing services or resources. This architecture enables efficient communication and scalability in various computing environments.

Frontend Development

Frontend development, or "front-end," is a web and software development discipline centered on creating visually appealing and user-friendly user interfaces (UI) and experiences (UX). Frontend developers focus on the client side, dealing with what users interact with in their browsers. A good UI effectively presents information, while UX encompasses the overall positive and efficient user interaction with a product. Web pages are composed of HTML for content structure, CSS for styling, and JavaScript for interactivity.
CSS layouts define the structure and arrangement of elements on a web page, determining their position, size, and relationships. In programming, languages like JavaScript act as a secret code to instruct computers, enabling users to teach their computers to perform cool and fun tasks by providing step-by-step instructions that the computer follows.

Class 6

Backend Development

Instagram's API (Application Programming Interface) is a set of protocols and tools that allows developers to interact with Instagram's backend services. It provides a way for external applications to access and utilize Instagram's features, such as retrieving user data, posting content, and interacting with the platform programmatically. The API enables third-party developers to integrate Instagram functionality into their applications, fostering a broader ecosystem of interconnected services and enhancing user experiences beyond the official Instagram platform.
Instagram's API utilizes common HTTP methods such as GET for data retrieval, POST for submitting content, and DELETE for removing resources. These methods enable developers to interact programmatically with Instagram's backend, facilitating actions like accessing user data, posting content, and managing interactions.

Class 7

Server Hosting and Deployment

A server is a system providing services to clients in a network, while hosting involves offering the infrastructure for websites or applications to be accessible online. Web hosting companies provide server space for storing website files, databases, and assets. Deployment is the act of making a software application or system available for use.

Server-side vs. client-side code

Client-side code executes on the user's computer, downloaded and run by the browser when a web page is viewed. It's responsible for the page's display. In contrast, server-side code runs on the server, and its outcomes are downloaded and displayed in the browser, handling processes on the server side. JavaScript is employed in various contexts, functioning both on the client side for interactive web elements and on the server side for backend operations.

Class 10

Data Storage

Data storage is the process of preserving and maintaining information for future use, utilizing various methods and technologies. This can include traditional options like hard disk drives and solid-state drives, as well as newer alternatives like cloud storage, tape storage, and optical storage.

Databases

It's an organized data collection meant for efficient storage, retrieval, and management, designed to facilitate streamlined querying and processing, while also enforcing relationships between different data elements.

Types of databases

Databases come in various types, each tailored to specific needs. Relational Databases (RDBMS) organize data into tables, enforcing relationships between them. Examples include MySQL, PostgreSQL, Oracle Database, and Microsoft SQL Server. NoSQL Databases prioritize flexibility and scalability, operating without a fixed schema. In-Memory Databases store data in the system's main memory (RAM) for quicker access. Distributed Databases enhance scalability and fault tolerance by distributing data across multiple nodes or servers. These are just a few examples, as various database types cater to different organizational, flexibility, and performance requirements.

Buzzwords – SQL

SQL (Structured Query Language) is a foundational tool for accessing and manipulating databases. Established as a standard by the American National Standards Institute in 1986, SQL empowers users to execute queries, retrieve data, and perform operations like inserting, updating, and deleting records within databases. Its capabilities extend to creating new databases, tables, stored procedures, and views, while also allowing users to set permissions on various database elements. The components of a SQL system include tables with rows and columns, SQL statements, and stored procedures.

The history of SQL dates back to its invention in the early 1970s by Donald D. Chamberlin and Raymond F. Boyce at IBM. Originally called SEQUEL (Structured English Query Language), it aimed to manipulate and retrieve data stored in IBM's original database management system, System R. Over the years, SQL has evolved into a versatile and widely adopted language, becoming a cornerstone in database management and data manipulation.

Version Control System

A Version Control System (VCS) is a software tool designed to facilitate the management of modifications to source code or any group of files over a period. It is widely employed in collaborative software development to monitor and synchronize changes made by various contributors working on a project.

GIT/ Github

Git is a decentralized version control system enabling collaboration among multiple developers on a project. GitHub, on the other hand, is a web-based platform offering Git repository hosting. It extends Git's capabilities through a graphical interface, collaboration tools, and supplementary features.
